About This Project

This is the WebC TUI Browser — a terminal-based web page renderer written in C++ from scratch. It parses real HTML and renders it beautifully right inside your terminal window.

No Electron. No browser engine. Just raw ANSI escape codes and box-drawing characters making the terminal feel alive.

Features

Supported Elements

  • All heading levels: h1 through h6
  • Paragraphs with bold and italic text
  • Unordered and ordered lists (nested too)
  • Tables with header rows
  • Code and preformatted blocks
  • Links, images, and forms
  • nav, header, footer, section, article

Upcoming

  1. CSS color parsing (inline styles)
  2. Mouse click support for links
  3. Scrolling for long pages
  4. JavaScript evaluation
