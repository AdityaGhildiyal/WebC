Code Sample

A fast in-place string reverse in C:

void str_reverse(char *s) {
    char *l = s;
    char *r = s + strlen(s) - 1;
    while (l < r) {
        char tmp = *l;
        *l++ = *r;
        *r-- = tmp;
    }
}
